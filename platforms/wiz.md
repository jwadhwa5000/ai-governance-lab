# Wiz for AI Security and CSPM

Evaluation and configuration guidelines for utilizing Wiz to discover shadow AI deployments, manage cloud security posture, and secure enterprise AI infrastructure.

## Core Capabilities

### 1. AI Security Posture Management (AI-SPM)
* **Shadow AI Discovery:** Automatically discover misconfigured or unapproved AI models, SDKs, and services across cloud environments.
* **Vulnerability Management:** Scan AI-specific container images, PyTorch/TensorFlow dependencies, and underlying cluster infrastructure.

### 2. Infrastructure & Integration CSPM
* **MCP Server Security:** Enforce posture management and secure baseline configurations for Model Context Protocol (MCP) servers to prevent unauthorized execution.
* **API & Interconnection Hardening:** Continuous configuration monitoring for APIs bridging AI orchestration layers to external observability platforms and enterprise data lakes.
* **Data Layer Protection:** Audit IAM policies, access controls, and encryption settings on object storage and data warehouses feeding training pipelines.
